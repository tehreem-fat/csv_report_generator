# CSV Report Generator

A Python project that generates summary reports and charts from CSV files.

## Features
- Reads CSV files and generates descriptive statistics
- Saves summary to Excel
- Generates charts for data visualization

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Add your CSV file in the `data/` folder
3. Run:
   python main.py
4. Check reports in the `reports/` folder
